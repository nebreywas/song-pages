Standing Rules on missing information
As Song Pages grows it is important we document how to import (and sometimes export) missing information. There are many reasons song information and related metadata can go missing. Bad imports, server timeouts, purposely left out information.

When you encounter missing information please speak up and ask how to handle it both in terms of storing the information but also presenting it to the user on screen or via exports. 

This document stands to help contain established rules where we can on how to handle various types of missing information:

Importing information about songs
How to handle missing information on imported tracks:

If a track title is missing replace the text with:
	Unknown Track Name

If an artist name is missing replace the text with:
	Name Unknown

If a song year is missing leave it blank

If an album name is missing leave it blank

If any expected URL is missing for a field that will store it to share again later just leave it blank

Missing information during VC Mode
We have already established some of these rules as part of our overall VC Mode Fallbacks plan that allows the host to customize how to handle the presentation of missing information in VCs.

Exporting information from Playlists
When exporting a playlist there could be reasons information has gone missing please handle it as follows:

If a track title is missing replace the text with:
	Unknown Track Name

If an artist name is missing replace the text with:
	Name Unknown

If a song year is missing leave it blank and remove the parentheses that would normally contain the information

If the song URL is missing leave it blank

If the song length is unavailable leave it blank
